# Security Policy

## Reporting a vulnerability

If you find a security issue, please do not open a public issue. Email stephen@botzone.ai with details.

We aim to respond within 48 hours and disclose responsibly.

## Scope

This repository contains the TypeScript Cost SDK. Vulnerabilities in the wrapper code, in the metadata payload, or in dependency handling are in scope.

The Cost dashboard (cost.botzone.ai) is a separate service. Issues there can also be reported via the same email.

## Out of scope

- Vulnerabilities in upstream provider SDKs (report to Anthropic, OpenAI, or Google directly)
- Issues that require physical access to a developer's machine
- Social engineering of project maintainers
